### Font Optimization
- Using `next/font/google` with CSS variables for optimized, self-hosted fonts
- Automatic font optimization at build time, reducing external dependencies
- CSS variable approach (`--font-dm-sans`) for flexible usage across the app
<hr>

### Next.js API Route Types
- Using `NextRequest` and `NextResponse` for type-safe API routes
- `NextRequest` provides typed access to searchParams, headers, and Next.js-specific features
- `NextResponse.json()` automatically handles JSON serialization and content-type headers
